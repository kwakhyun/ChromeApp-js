## Vanilla To-do list

### 주요 기능
- 할 일 메모 및 삭제
- 로컬 스토리지를 이용한 데이터 저장
- 현재 시간, 날씨 정보
- 랜덤 명언

### 프로젝트 소개
로컬 환경에서 사용하기 좋은 할 일 메모 앱입니다. 오픈 API를 이용한 날씨 정보를 제공하며, 페이지 렌더링 시 랜덤한 명언을 보여줍니다.

### 사용 스택
`HTML` `CSS` `JavaScript`
